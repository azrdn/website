# my website

![Node.js >= 22](https://img.shields.io/badge/node.js-%3E%3D22-brightgreen)
![pnpm >= 11](https://img.shields.io/badge/pnpm-%3E%3D11-blue) 

Hybrid SSR/SSG site using [Astro](https://astro.build/) and [Hono](https://hono.dev/)

Requirements:
- Node.js
- PNPM

## develop
- fill the .env file
- run `pnpm dev`
- go wild

## deploy
- fill the .env file
- run `pnpm build`
- results in `dist` directory in each workspace
- and then you go figure it out yourself
