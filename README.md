# my website
[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2Fazrdn%2Fwebsite.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2Fazrdn%2Fwebsite?ref=badge_shield)


Hybrid SSR/SSG site using [Astro](https://astro.build/) and [Hono](https://hono.dev/)

Requirements:
- Node.js
- PNPM

## develop
- fill the .env file
- run `pnpm dev`
- go wild

## deploy
- fill the .env file
- run `pnpm build`
- results in `dist` directory in each workspace
- and then you go figure it out yourself
